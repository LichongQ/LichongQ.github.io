A. 递归实现

在反转当前节点时，先反转当前节点的下一个节点。直至链表的最后一个节点，才开始反转链表。即利用递归的特性，从链表的尾节点开始进行链表的逆序。

B. 遍历实现（非递归）

按照链表的顺序，从头结点开始依次反转各节点。
